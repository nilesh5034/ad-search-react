# Ad Search API - ASP.NET Core Backend

RESTful API for Ad Search application built with ASP.NET Core 8.

## 🚀 Quick Start

```bash
cd Backend
dotnet restore
dotnet run
```

API runs on: `https://localhost:5001`
Swagger UI: `https://localhost:5001/swagger`

## 📡 API Endpoints

### Get All Ads
```
GET /api/ads
```

### Search Ads
```
GET /api/ads/search?searchTerm=laptop
```

### Get User Details with Ads
```
GET /api/users/{id}
```
