# Ad Search React Application

A full-stack **Ad Search Application** built with **React** frontend and **ASP.NET Core** backend API.

## 📁 Project Structure

```
ad-search-react/
├── Backend/          # ASP.NET Core 8 Web API
│   ├── Controllers/
│   ├── Models/
│   ├── Services/
│   ├── Program.cs
│   └── AdSearchAPI.csproj
├── Frontend/         # React Application
│   ├── src/
│   │   ├── components/
│   │   ├── pages/
│   │   ├── services/
│   │   ├── App.tsx
│   │   └── index.tsx
│   ├── package.json
│   └── tsconfig.json
└── README.md
```

## 🚀 Quick Start

### Backend Setup (ASP.NET Core)

```bash
cd Backend
dotnet restore
dotnet run
```

Backend runs on: `https://localhost:5001`

### Frontend Setup (React)

```bash
cd Frontend
npm install
npm start
```

Frontend runs on: `http://localhost:3000`
