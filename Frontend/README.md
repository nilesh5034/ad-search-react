# Ad Search React - Frontend

React frontend application for Ad Search using TypeScript.

## 🚀 Quick Start

```bash
cd Frontend
npm install
npm start
```

Frontend runs on: `http://localhost:3000`

## 📡 API Configuration

The app connects to the backend API at: `https://localhost:5001`

Make sure the backend is running before starting the frontend.

## 📖 Features

✅ Real-time ad search  
✅ View seller profiles  
✅ Display seller's ads  
✅ Responsive design
